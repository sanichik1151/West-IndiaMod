# Гайд по установке
Скопируйте установленный архив в ```Документы\Paradox Interactive\Hearts of Iron IV\mod.``` 
Скопируйте файл descriptor.mod из папки мода в папку mod (на уровень выше) и переименуйте его в modwestindia.mod. Откройте modwestindia.mod и добавьте в конец строку: 
```path="C:/Users/ВашеИмя/Documents/Paradox Interactive/Hearts of Iron IV/mod/modwestindia"```
